### Request-Response Flow:
User types "What is React?" вҶ’ Clicks SEND
React (Frontend) вҶ’ axios.POST вҶ’ http://localhost:4002/bot/v1/message
Express (Backend) вҶ’ Receives { text: "What is React?" }
Controller вҶ’ Saves user message to MongoDB (Users collection)
Controller вҶ’ Matches text with botResponses object
Controller вҶ’ Finds response for "what is react"
Controller вҶ’ Saves bot response to MongoDB (Bots collection)
Controller вҶ’ Returns JSON { userMessage, botMessage }
React (Frontend) вҶ’ Receives response
React вҶ’ Shows "thinking" animation for 1.5-3 seconds
React вҶ’ Displays bot response with timestamp

Go to MongoDB Atlas
Create a free account

Copy the connection string
